# Hono API with Deno

## Quickstart

```bash
deno cache *.ts # fetch dependencies
deno run --allow-all --watch main.ts # run in development mode
```

## Test

```bash
deno test -A
```

## Deploy

Create an account at https://deno.com/deploy, link your repository and go to URL
generated for you. https://system32uwu-hono-deno.deno.dev/
